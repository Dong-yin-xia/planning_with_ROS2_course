### 目录：decision_center（决策中心）

**功能概述**
- 根据车辆与障碍物在参考线坐标系下的状态，生成 SL 关键点（变道/停止/起止点），供局部路径规划使用。

**核心类**
- `DecisionCenter`
  - 读取 `decision`、`pnc_map`、`refer_line`、`local_path` 等配置。
  - `make_path_decision(car, obses)`: 针对每个满足条件的障碍物，计算左右可通行带宽；若可通行生成 `LEFT_PASS/RIGHT_PASS` 点；两侧都不满足则生成 `STOP` 点；并在首尾增加 `START/END` 形成完整段。
  - 输出 `sl_points()`：有序 SL 断点序列。

**实现方式**
- 宽度判据：对障碍物左右侧可用宽度与车辆宽度 + 2×侧向安全距离进行比较。
- 距离与范围：参考线前向窗口、最小变道距离、考虑障碍物范围由 YAML 参数给定。
- 复杂度可控：当前逻辑偏启发式，便于教学与调试，可扩展为多障碍优先级/博弈/代价最小化策略。
